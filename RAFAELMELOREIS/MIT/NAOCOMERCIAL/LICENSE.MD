# Pré-paper v0.1

## Matriz Bagua-Toroidal de Compressão, Expansão e Centro Observador

### Resumo

Este texto organiza uma hipótese geométrico-dinâmica baseada na relação entre yin-yang, bagua/paqua, triângulos equiláteros, quadrado, octógono, centro observador, fluxo toroidal, compressão-expansão e ciclos discretos de decisão. A tese não é apresentada como prova física literal, mas como um **modelo heurístico formalizável**: uma estrutura para transformar imagens simbólicas em geometria, matriz, campo vetorial, grafo e possível simulação. O núcleo do modelo já foi sintetizado como a ideia de que “opostos giratórios criam compressão, expansão, áreas faltantes e pontos de estabilidade”, gerando triângulos, círculos, quadrados, hexágonos, toroides e padrões de fluxo. 

### Palavras-chave

Yin-yang; bagua; toroide; octógono; triângulo equilátero; √3/2; Venturi; centro observador; matriz 10×10×10; grafo 42; ciclo 52; heurística geométrica.

---

## 1. Problema

A questão central é:

**Como representar oposição, rotação, compressão, expansão e decisão dentro de uma geometria mínima que passe de 2D para 3D sem perder coerência?**

O modelo parte de quatro intuições:

1. O yin-yang não é apenas símbolo, mas uma representação de oposição dinâmica.
2. O bagua discretiza essa oposição em oito direções.
3. O centro funciona como observador, eixo de decisão ou ponto de colapso.
4. O triângulo estabiliza a oposição entre dois polos.

## 2. Base conceitual

### 2.1 Oposição primária

A base não é “quatro coisas”. A base é o ato mínimo:

| Estado mínimo | Leitura                           |
| ------------- | --------------------------------- |
| ter           | presença                          |
| não ter       | ausência                          |
| 0             | não manifestado                   |
| 1             | manifestado                       |
| reta          | direção entre presença e ausência |

No caso do qubit, o ponto técnico correto é: ele possui estados-base |0⟩ e |1⟩, mas também pode existir em superposição α|0⟩ + β|1⟩ antes da medição. Portanto, a tua leitura “não começa em 4; começa no ter/não ter” é boa como intuição binária, desde que não seja confundida com a matemática completa do qubit. ([Microsoft Quantum][1])

### 2.2 Redução escalar

| Escala | Forma                  | Função no modelo     |
| ------ | ---------------------- | -------------------- |
| 1      | ponto                  | existência           |
| 2      | reta                   | oposição             |
| 3      | triângulo              | estabilização        |
| 4      | quadrado               | campo fechado        |
| 8      | octógono/bagua         | decisão direcional   |
| 9      | 8 + centro             | matriz de observação |
| 10     | 8 + 2 núcleos internos | yin-yang expandido   |
| 42     | grafo de transições    | ciclo/atratores      |
| 52     | ciclo maior            | retorno temporal     |

O ciclo 52 possui uma referência histórica real no calendário maia/mesoamericano: a combinação entre o ciclo ritual de 260 dias e o ano de 365 dias gera um ciclo de 18.980 dias, ou 52 anos, chamado Calendar Round. ([Encyclopedia Britannica][2])

## 3. Hipótese central

A hipótese pode ser escrita assim:

**Um campo de oposição recíproca, quando projetado sobre uma matriz quadrangular e submetido à rotação, gera áreas visíveis, áreas faltantes, eixos medianos, triângulos de estabilização e regiões curvas. A decisão emerge quando a oposição encontra centro, eixo ou terceira relação.**

Em forma estrutural:

| Elemento             | Papel                                |
| -------------------- | ------------------------------------ |
| yin                  | contração                            |
| yang                 | expansão                             |
| pontos internos      | memória da oposição dentro do oposto |

| octógono             | oito decisões                        |
| triângulo equilátero | estabilidade                         |

## 4. Geometria mínima

### 4.1 Triângulo equilátero

A constante geométrica principal é a altura do triângulo equilátero:

**h = √3 / 2**

Ela entra como constante de:

* projeção;
* estabilidade;
* contração;
* transição plano-centro;
* proporção entre base e altura.

### 4.2 Triângulo isósceles

O triângulo isósceles aparece quando o sistema não está perfeitamente equilátero. Ele representa:

| Situação           | Interpretação      |
| ------------------ | ------------------ |
| lados iguais       | simetria parcial   |
| altura deslocada   | tensão             |
| base incompleta    | área faltante      |
| retorno ao centro  | estabilização      |
| fechamento de fase | determinismo local |

Aqui está uma parte forte da tese: **o triângulo isósceles não é o começo; ele é o resultado de uma oposição tentando estabilizar.**

### 4.3 Quadrado e octógono

O quadrado define o campo mínimo fechado.

O octógono surge quando:

* o quadrado recebe diagonais;
* as quatro direções viram oito;
* o giro cria estados intermediários;
* o centro passa a funcionar como ponto 9.

Então:

**4 direções → 8 decisões → 9 com centro → 10 com os dois núcleos internos.**

## 5. Modelo yin-yang expandido

No yin-yang clássico, cada lado contém o ponto do outro. A tua expansão propõe:

| Parte                                | Leitura                                         |
| ------------------------------------ | ----------------------------------------------- |
| grande por fora / pequeno por dentro | força dominante carregando semente oposta       |
| pequeno por fora / grande por dentro | força aparente menor com potência interna maior |
| duas bolinhas internas               | dois núcleos de memória/oposição                |
| giro                                 | alternância de domínio                          |
| gota                                 | forma de transição entre compressão e expansão  |
| centro                               | decisão/observação                              |

Essa leitura já estava presente no prompt anterior: as “bolinhas” internas foram descritas como núcleos recíprocos de igual tamanho girando entre si, enquanto o corpo maior e menor expressa compressão e expansão.

## 6. Área faltante e antiderivada conceitual

### 6.1 Antiderivada

Aqui, “antiderivada” pode ser entendida como a causa geradora antes da forma aparecer.

| Antiderivada            | Derivada geométrica |
| ----------------------- | ------------------- |
| oposição                | reta                |
| dupla oposição          | cruz/quadrado       |
| rotação da oposição     | círculo             |
| tensão reta-curva       | área faltante       |
| área faltante           | parábola/arco       |
| arco repetido           | pétala/mandala      |
| pétala em campo fechado | octógono            |
| octógono com centro     | decisão             |

### 6.2 Área que não existe

A “área que não existe” é o espaço usado para fechar a conta entre:

* reta;
* arco;
* tangente;
* parábola;
* triângulo incompleto;
* curva dobrada para dentro.

Formalmente, isso pode ser tratado como:

**A_falta = A_triângulo projetado − A_curva ocupada**

Não precisa calcular agora. O importante é definir onde medir.

## 7. Toroide, Venturi e pulso

O modelo 3D sobe assim:

| 2D           | Rotação         | 3D                                 |
| ------------ | --------------- | ---------------------------------- |
| círculo      | eixo            | esfera                             |
| quadrado     | eixo            | cilindro aproximado/cubo projetivo |
| triângulo    | eixo            | cone                               |
| arco fechado | eixo duplo      | toroide                            |
| octógono     | rotação angular | malha/anel direcional              |

O efeito Venturi entra como analogia técnica: quando um fluxo passa por uma região mais estreita, sua velocidade e pressão mudam conforme a dinâmica de escoamento descrita por Bernoulli/Venturi. Use isso como analogia de compressão-expansão, não como prova direta de matéria. ([Microsoft Quantum][1])

No teu modelo:

| Fase       | Descrição                                |
| ---------- | ---------------------------------------- |
| contração  | fluxo entra no centro                    |
| aceleração | região estreita aumenta tensão           |
| expansão   | fluxo sai para borda                     |
| retorno    | campo recircula                          |
| pulso      | diferença entre mínimo, máximo e mediana |

## 8. Matéria, força e reação

A parte da parede devolvendo o soco está alinhada com a ideia clássica de ação e reação: uma força aplicada encontra uma força oposta. No modelo, isso vira:

**matéria = campo que responde à tentativa de atravessamento.**

Mas a parte da bomba atômica precisa ficar separada:

* fisicamente, parte da massa é convertida em energia;
* a relação base é E = mc²;
* não é correto dizer literalmente que “a gravidade faltou”;
* como analogia, pode-se dizer: massa condensada → energia liberada → pulso expansivo.

## 12. Formulação curta da tese

**Propõe-se um modelo geométrico no qual a oposição yin-yang é tratada como um sistema rotacional de presença e ausência. Quando projetada em matriz quadrangular, essa oposição gera quatro quadrantes, oito direções, um centro observador e dois núcleos internos de oposição recíproca. A passagem entre reta e curva produz áreas faltantes mensuráveis, estabilizadas por triângulos equiláteros e isósceles. Em 3D, o modelo assume forma toroidal, onde compressão e expansão podem ser lidas como pulsos de entrada, saída e retorno.**
